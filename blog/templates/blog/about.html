About Us

This is a simple Twitter clone made to emulate the old-style Twitter design. It's a minimal, lightweight mock-up, created as a demonstration of basic web design using HTML and CSS.

It includes a home page for a timeline, a profile section, and a tweet input. The About page gives a brief description of the project.

Features:

  • Profile Section with Bio
  • Basic Timeline with Tweets
  • Simple Tweet Box
  • Navigation between Home and About pages
